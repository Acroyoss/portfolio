Table Tags

Tags Name Description
<table> Create Table Table elements will go inside this tags
<tr> Table Row This will create a row for you to add data
<td> Table Data Use it inside <tr> to add data into a table
<thead> Table Head To define the header of the table
<th> Table Heading Use to add data inside the <thead>
rowspan Rowspan Attribute This will set how many rows to span
colspan Colspan Attribute This will set how many columns to span
<tbody> Table Body To define the body of the table
<tfooter> Table Footer Element uses table rows to give footer content or to summarize content at the end of a table
