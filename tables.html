Table Tags

Tags Name Description
<table> Create Table Table elements will go inside this tags
<tr> Table Row This will create a row for you to add data
<td> Table Data Use it inside <tr> to add data into a table
<thead> Table Head To define the header of the table
<th> Table Heading Use to add data inside the <thead>
rowspan Rowspan Attribute This will set how many rows to span
colspan Colspan Attribute This will set how many columns to span
<tbody> Table Body To define the body of the table
<tfooter> Table Footer Element uses table rows to give footer content or to summarize content at the end of a table

Elements and Structures

Tags Name Description
<li> List Items create list items inside the tags below
<ul> Unordered List A non-numeric list
<ol> Ordered List A numeric list
<video> Video Element embeds a media player for video playback. The src attribute will contain the URL to the video. Adding the controls attribute will display video controls in the media player.
<div> Div Element used as a container that divides an HTML document into sections and is short for “division”.
<img> Image Element Use src attribute to contain the image URL and is mandatory. It is an empty element meaning it should not have a closing tag.
<h1> <h6> Heading Elements The heading elements are ordered from the highest level <h1> to the lowest level <h6>
<p> Paragraph Element paragraph element contains and displays a block of text.
<body> Body Element Content inside <body> tags are rendered on the web browsers.
